# Calendario-Curso
Calendario de cursos de matemática


- Este calendario en específico está hecho para FOKITO, pero puede ser modificado para aplicarse a cualquier otro curso modificando calendario_config.yml

Archivos:
- app.py: corre la app web
- generador_excel.py: genera un excel con las reglas impuestas por calendario_config.yml
- calendario_config.yml: configuraciones específicas para el curso de FOKITO, se irá modificando durante el semestre
